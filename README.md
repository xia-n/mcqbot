# Discord Quiz Bot

This repository contains a Discord quiz bot built using `discord.py`.  
The bot allows users to participate in quiz sessions, tracks scores, and supports easy customization through a JSON-based database.

---

## Overview

The bot provides both solo and competitive quiz modes with automatic score tracking.  
It is designed to work within a designated channel to maintain order and prevent spam.  
All data, including questions, answers, and scores, are stored in a `database.json` file.

---

## Features

- Solo quiz mode with random question selection  
- Automatic score tracking and persistence  
- JSON-based storage for questions and answers  
- Configurable quiz channel restriction  
- Built-in debugging messages for testing and verification

---

## Requirements

- Python 3.10 or later  
- `discord.py` library  
- A valid Discord bot token

To install the required library, run:

```bash
pip install discord.py
```
